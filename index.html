Profil auf diesem Gerät

Lokale Daten löschen

Diese Aktion entfernt alle FitTrack-Einträge aus diesem Browser. Ein Backup kann danach wieder importiert werden.

Was „lokal“ bedeutet

  • Die Daten liegen nur in diesem Browser auf diesem Gerät.
  • Andere Geräte werden nicht automatisch synchronisiert.
  • Das Löschen der Website-Daten kann auch FitTrack-Daten entfernen.
  • Die App funktioniert nach dem ersten vollständigen Laden auch ohne Internet.
